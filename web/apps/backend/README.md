# Backend

Backend project_zen udostępnia REST API dla panelu administracyjnego, endpointy dla przyszłego Android Agent, Socket.IO dla odświeżania realtime oraz Prisma ORM dla PostgreSQL.

## Komendy

```bash
npm run dev -w @project_zen/backend
npm run db:generate -w @project_zen/backend
npm run db:dev -w @project_zen/backend
npm run db:seed -w @project_zen/backend
```

Swagger jest dostępny pod `http://localhost:4000/api/docs`.
